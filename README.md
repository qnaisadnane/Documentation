---------------------------- Categorie 1 -----------------------------

var : variable qui peut etre appeler dans toute la fonction et on peut reécrire var avec le meme nom du variable por affecter un valeur

let : variable qui peut etre appeler seulement dans son bloc et on peut pas reécrire let avec le meme nom du variable por affecter un valeur

const : constante qui ne peut pas changer sa valeur
