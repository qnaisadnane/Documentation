-------------variable-------------------

-----------------difference-------------

var : déclarer un  variable qui peut etre appeler dans toute la fonction et on peut reécrire var avec le meme nom du variable por affecter un valeur

let :déclarer un variable qui peut etre appeler seulement dans son bloc et on peut pas reécrire let avec le meme nom du variable por affecter un valeur

const : constante qui ne peut pas changer sa valeur

---------------utilisation---------------

 * on utilise const pour creer un constant qui ne peut pas etre changé

exemple : const nom = document.getElementbyId('id');

 * var pour un déclarer un variable globale

var x =3;
var x = 6;

 * let pour déclarer un variable seulement dans les blocs

 let a = 3
